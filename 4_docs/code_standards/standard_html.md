# HTML命名规范
### 一、代码格式
- class模块  
  驼峰式 + 删除线  
  
	```
		<div class"""=navMain-cont><div>
	```
	
- ID模块  
  驼峰式 + 下划线

	```
		<div id"""=navMain_cont><div>
	```
